Guide d'utilisation


  • ETAPE 1

    INSCRIPTION

    Une fois sur la page d'accueil, allez sur l'onglet "Inscription". Remplissez le premier formulaire avec les informations requises (nom, prénom, adresse e-mail, etc.). Suivez les instructions pour créer un compte.


  • Etape 2

    Informations sur votre entreprise

    Après avoir rempli le formulaire d'inscription, un second formulaire s'affichera. Remplissez ce formulaire avec les informations de votre entreprise (nom de l'entreprise, adresse, numéro de téléphone, etc.). Vérifiez bien toutes les informations fournies avant de les soumettre.

    Une fois que vous avez terminé de remplir tous les formulaires, notez votre identifiant (adresse e-mail) et le mot de passe que vous avez choisis. Cela vous permettra de vous connecter ultérieurement. Vous recevrez également un message indiquant que vos informations ont été ajoutées avec succès. Consultez votre e-mail pour une confirmation de votre inscription. Si vous ne recevez pas cet e-mail, vérifiez votre dossier de spam.


  • Etape 3

    Postuler pour un marché

    Après vous être inscrit(e), accédez à l'accueil et vous verrez la section des marchés publics. Recherchez le marché xxxxxx pour lequel vous souhaitez postuler. Cliquez sur "Postuler" pour ce marché.

    Suivez les instructions pour téléverser les documents demandés. Une fois les documents sont téléversés, cliquez sur "Je postule" pour valider votre candidature.

    Après la soumission, Vérifiez vos e-mails pour une confirmation de votre candidature. Assurez-vous que vous avez bien reçu une confirmation pour valider que votre candidature a été soumise avec succès.


  • Etape 4

    Voir L'etat de vos candidatures

    Cliquer sur l'onglet (mes candidatures) pour voir l'evolution de vos candidatures

    Conseil Supplementaire

    Notez bien votre mot de passe lors de la création de votre compte. Assurez-vous que tous les documents nécessaires sont prêts avant de commencer le processus de candidature. Si vous avez des questions ou des problèmes, consultez la section guide de la plateforme ou contactez le support technique. En suivant ces étapes, vous devriez être en mesure de vous inscrire, de renseigner les informations sur votre entreprise, de postuler pour un marché et de vérifier vos candidatures sans difficulté
